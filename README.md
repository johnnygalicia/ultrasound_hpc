# Ultrasound HPC

Proyecto final de Computación Paralela aplicado a procesamiento de imágenes médicas por ultrasonido.

## Tecnologías
- Python
- OpenCV
- OpenCL
- PyOpenCL
- AMD GPU

## Objetivos
- Reducir ruido speckle
- Procesar video médico por frames
- Comparar CPU vs GPU
- Implementar filtros paralelos

## Resultados
- Bilateral filtering
- CLAHE enhancement
- Benchmark CPU/GPU
- Reconstrucción de video
